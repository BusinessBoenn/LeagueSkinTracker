## Prerequisites
psutil, flask, [lcu-driver](https://github.com/sousa-andre/lcu-driver)
```
pip install psutil flask lcu-driver
```

# Usage
### First time use:
Make sure to have League Client open to create needed file <br />


### General:
Run app.py<br />
Open League Client to update owned skins via the button in the top right corner (optional) <br />
